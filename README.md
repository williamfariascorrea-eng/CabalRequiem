## 🚀 Quick Start

```bash
# Clone o projeto
git clone https://github.com/williamfariascorrea-eng/CabalRequiem.git

# Abra no navegador
cd CabalRequiem
start index.html
```

Ou simplesmente abra o arquivo `index.html` no seu navegador.
